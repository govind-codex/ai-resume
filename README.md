# AI Interviewprep &  Strategy Planner

A modern web application that analyzes job descriptions and user profiles/resumes to generate customized interview preparation plans, including targeted technical questions, behavioral strategies, and localized skill-gap analyses.

## Key Features
 🧠 AI Analysis: Leverages AI to map your profile directly against target job requirements.
 📋 Custom Roadmaps: Generates step-by-step preparation timelines.
 💬 Practice Questions: Provides tailored technical and behavioral questions with suggested answers.
 🪟 Modern Interface: Beautiful ocean-breeze inspired UI featuring accessible dark modes and polished glassmorphic auth screens.

## Quick Start

### Backend
1. Go to `/Backend`
2. Create `.env` and configure port and MongoDB URI.
3. Run `npm install` followed by `npm run dev`.

### Frontend
1. Go to `/Frontend`
2. Run `npm install` followed by `npm run dev`.
3. Open `http://localhost:5173`.
